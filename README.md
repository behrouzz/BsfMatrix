# BsfMatrix

## Installation

To install this package, import `url here` in Swift Package Manager.

## Usage example

```swift
let a = [
    [2.0, 5.0, 4.0],
    [8.0, 1.0, 3.0],
    [3.0, 2.0, 6.0],
    [5.0, 4.0, 7.0],
    [2.0, 7.0, 8.0],
]

let b = [
    [2.0, 2.0, 2.0, 1.0, 5.0],
    [1.0, 7.0, 2.0, 6.0, 3.0],
    [5.0, 3.0, 9.0, 2.0, 4.0]
]

print(a[0] * a[1])
```
